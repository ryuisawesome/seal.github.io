Discord Utility Bot

Seal is a multi-purpose discord bot that has various purposes and commands to make sure that your discord bot use is easy.

Add to Discord Discord Server

Roblox User Commands

User Information

  • /user - Get detailed information about a Roblox user
  • /avatar - Get a user's Roblox avatar
  • /headshot - Get a user's Roblox headshot
  • /names - Get a user's username history
  • /currentlywearing - Fetch items a user is wearing
  • /outfits - Get all outfits for a user
  • /followers - Get a user's followers
  • /following - Get who a user is following
  • /friends - Get a user's friends

Group Information

  • /group - Get detailed group information
  • /groupicon - Get a group's icon/emblem
  • /ownedgroups - Get groups owned by a user
  • /currentgroups - Get groups a user is in

Miscellanous

  • /item - Get detailed item information
  • /placeinfo - Get information about a Roblox place
  • /verified - Check if a user owns the verified hats
  • /recentbadges - Get a user's most recently earned badges

Minecraft Commands

Player Information

  • /minecraft - Get Minecraft player information
  • /minecraft_bulk - Check multiple Minecraft usernames

Utility Commands

Validation

  • /validate - Check if a username is available on Roblox

Investigation Tools

  • /iplookup - Get IP address information
  • /github - Get GitHub user information
  • /instagram - Get Instagram user information

Discord Utilities

  • /info - Get Discord user information
  • /help - Show help menu
  • /notes - Important information about the bot
  • /timestamp - Convert ISO timestamp to Discord format
